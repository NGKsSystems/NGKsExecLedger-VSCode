# NGKs Autonomous Logger (Phase 1)

Phase 1 includes:
- Session lifecycle (auto start)
- JSONL event writer with hash chaining
- Extension-host exception hooks
- Autosaved proof copy to Downloads

Commands:
- NGKs: Start Logging Session
- NGKs: Stop Logging Session

Settings:
- ngksAutologger.enabled
- ngksAutologger.logRootMode (workspace/global)
- ngksAutologger.autosaveToDownloads
- ngksAutologger.appName
